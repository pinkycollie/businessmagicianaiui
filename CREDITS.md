# Credits and Acknowledgments

## Original Project

This project is a fork of **shadcn/ui**, an outstanding open-source component library.

### shadcn/ui
- **Creator**: [shadcn](https://twitter.com/shadcn)
- **Repository**: [github.com/shadcn-ui/ui](https://github.com/shadcn-ui/ui)
- **Website**: [ui.shadcn.com](https://ui.shadcn.com)

All core components, architecture, and design patterns in this project originate from shadcn/ui. We are grateful for the excellent work and the open-source nature of the project.

## Core Technologies

### Radix UI
- **Website**: [radix-ui.com](https://www.radix-ui.com/)
- **Description**: Unstyled, accessible component primitives for React
- **License**: MIT

Radix UI provides the foundation for accessible components with:
- WAI-ARIA compliant patterns
- Keyboard navigation
- Focus management
- Screen reader support

### Tailwind CSS
- **Website**: [tailwindcss.com](https://tailwindcss.com/)
- **Creator**: Adam Wathan and Tailwind Labs
- **Description**: A utility-first CSS framework
- **License**: MIT

Tailwind CSS powers the styling system with:
- Utility-first approach
- Customizable design system
- Dark mode support
- Responsive design utilities

### tailwindcss-animate
- **Repository**: [github.com/jamiebuilds/tailwindcss-animate](https://github.com/jamiebuilds/tailwindcss-animate)
- **Description**: Animation utilities for Tailwind CSS
- **License**: MIT

### Supporting Libraries

- **class-variance-authority (CVA)**: Component variant management
- **clsx**: Utility for constructing className strings
- **tailwind-merge**: Intelligently merge Tailwind CSS classes
- **next-themes**: Theme management for Next.js
- **Lucide React**: Beautiful and consistent icon set
- **React Hook Form**: Performant form validation
- **Zod**: TypeScript-first schema validation
